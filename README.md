# QA-Snake
![Eric](https://raw.githubusercontent.com/SnakeHacker/QA-Snake/master/resources/Eric-Logo.png)
## QA-Snake是什么?
目前QA-Snake是一个基于搜索引擎的自动问答机器人（Eric）

## Eric有哪些功能？

* 支持问句类型有：
    *  问人
    *  问地点
    *  地图查询
* 闲聊
* 运价查询

## 使用方法
   直接运行MainProgram.py
   目前只支持命令行模式
