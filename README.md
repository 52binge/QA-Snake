# QA-Snake是什么?
目前QA-Snake是一个基于多搜索引擎的自动问答机器人（Eric）

# Eric有哪些功能？

* 问答
* 闲聊
* 运价查询（后期要做）

# 使用方法
   测试环境为Python2.7(Anaconda2)
   需要额外安装的Python包有：
   * pip install aiml
   * pip install beautifulsoup4
   ## 直接运行MainProgram.py
   目前只支持命令行模式和Socket模式。
